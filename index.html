Globolecciones
Una forma divertida de ver los resultados del Balotaje.

- / - | Actualizar datos

Globolecciones es un proyecto personal para incursionar un poco en la visualización de datos y jugar con svg animado por css y js. La idea es aprovechar las APIs públicas de las elecciones para hacer algo diferente. Ningún partido político o gobierno está involucrado en el desarrollo o la idea.

El progreso de inflado es proporcional a la cantidad de votos de cada partido, y al porcentaje de mesas escrutadas al momento. Una vez superado el 94% de mesas escrutadas, el que saque mayor porcentaje lo demostrará con alegría.

Las animaciones las ejecuta el procesador, por lo que no te asustes si levanta un poco de temperatura o si el cooler se convierte en un pequeño tornado.
